| Name          | Value                                    | 是否必填 |
| ------------- | ---------------------------------------- | -------- |

| SERVERPUSHKEY | server酱推送的sckey                      | 必填     |

***DEVICENAME***变量填写例子：

例如：完整mac为：ABC12D456EF7(取后6位)    想设置为韭菜1号

单个设备：456EF7:韭菜1号

多个设备：456EF7:韭菜1号&789FE1:韭菜2号&123FR2:韭菜3号             （中间使用&连接即可）
